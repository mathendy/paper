README: LoopSubdivision

A simple m file for the triangle mesh subdivision using the Loop's scheme.
The zip file includes four examples of subdivision.

Author:
    Jesus Mena

References:

[2] "Subdivision for Modeling and Animation".
    SIGGRAPH 2000 Course Notes.
    http://www.cs.nyu.edu/~dzorin/sig00course/

Keywords:
